tags.scss
====
### wuts a tags.scss?
**tags.scss** is a mixin for styling up those tag things you see all over the interweb.

### Browser support
##### Full compat
- Chrome & Firefox
- IE9+

##### Partial compat
- IE6-8

Partial compat refers to a lack of a generated circle for the tag.
